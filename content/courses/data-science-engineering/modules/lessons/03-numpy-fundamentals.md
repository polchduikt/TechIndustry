## 🧠 Теорія
NumPy дає швидкі масиви і векторні операції для обчислень. На великих даних векторизація суттєво швидша за Python цикли. Це база для подальших ML та аналітичних обчислень.

### Поглиблено
У Data Science стабільний результат дає дисципліна процесу: чітка постановка задачі, відтворювані експерименти, контроль ризиків data leakage та постійний моніторинг після релізу.

### Приклад 1
Операції: mean, std, matrix multiplication для числових ознак.

### Приклад 2
Практика: переходити від поелементних циклів до операцій над усім масивом.
