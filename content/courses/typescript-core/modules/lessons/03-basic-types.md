# Базові типи

Базові типи — фундамент TypeScript, який дозволяє задавати чіткі очікування для даних у коді.

---

## 🎯 Мета уроку
Після цього уроку ти зможеш:
- використовувати примітивні типи
- типізувати масиви і кортежі
- розуміти `any`, `unknown`, `never`
- обирати безпечніші типові рішення

---

## 🧠 Теорія

### Примітиви
```ts
let name: string = "Olha";
let age: number = 23;
let active: boolean = true;
```

---

### Масиви і кортежі
```ts
const skills: string[] = ["TS", "React"];
const point: [number, number] = [10, 20];
```

---

### `any` і `unknown`
```ts
let a: any = 10;
let u: unknown = "text";

if (typeof u === "string") {
  console.log(u.toUpperCase());
}
```

---
### Глибше про якість типів
Базові типи — це перший рівень захисту, але справжня цінність з'являється, коли типи відповідають бізнес-сутностям. Наприклад, краще мати окремий тип `UserId`, ніж всюди використовувати абстрактний `number`.

Чим ближче типи до предметної області, тим швидше знаходяться помилки, пов'язані з неправильним змішуванням даних.

## 💻 Практичний приклад

```ts
type UserRole = "student" | "mentor";

interface User {
  id: number;
  name: string;
  role: UserRole;
}

const user: User = { id: 1, name: "Ірина", role: "student" };
console.log(user);
```

---

## 📝 Підсумки
- правильні базові типи знижують кількість логічних багів
- `unknown` безпечніший за `any`
- unions дозволяють описувати реальні доменні значення
