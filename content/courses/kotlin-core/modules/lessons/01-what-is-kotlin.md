# Що таке Kotlin

Kotlin — сучасна статично типізована мова програмування від JetBrains. Вона поєднує лаконічний синтаксис, безпечну роботу з `null` та повну сумісність з Java-екосистемою.

---

## 🎯 Мета уроку
Після цього уроку ти зможеш:
- пояснити, чому Kotlin став популярним
- назвати ключові переваги Kotlin для командної розробки
- розуміти, де саме Kotlin застосовується на практиці
- відрізняти Kotlin-підхід від класичного Java-підходу

---

## 🧠 Теорія

### Що робить Kotlin сильним
- короткий і читабельний синтаксис
- вбудований `null safety`
- підтримка функціональних та ООП-патернів
- проста інтеграція з Java-бібліотеками

```kotlin
val language = "Kotlin"
println("Мова: $language")
```

---

### Де Kotlin використовується
- Android-додатки
- Backend (Ktor, Spring)
- Multiplatform-проєкти
- Інструменти автоматизації

```kotlin
fun platformInfo(platform: String): String {
    return "Kotlin працює для: $platform"
}
```

---

### Kotlin і Java в одному проєкті
Kotlin можна додавати поступово: нові модулі пишуться на Kotlin, а старі Java-модулі залишаються працювати без переписування.

```kotlin
// Kotlin-код може викликати Java-класи без спеціальних адаптерів
val list = java.util.ArrayList<String>()
list.add("Interop")
println(list)
```

---
### Глибше про екосистему Kotlin
Kotlin розвивається як універсальна мова: для Android, серверів, клієнтів і спільної бізнес-логіки між платформами. На практиці це означає, що команда може використовувати спільні підходи до архітектури, стилю коду та тестування, навіть якщо продукт має кілька клієнтів.

Крім синтаксису, Kotlin цінують за передбачувану поведінку та хорошу інтеграцію з інструментами JetBrains. Це зменшує витрати на підтримку коду в довгострокових проєктах.

## 💻 Практичний приклад

```kotlin
data class Course(val title: String, val hours: Int)

fun main() {
    val course = Course("Kotlin Core", 24)
    println("Курс: ${course.title}")
    println("Тривалість: ${course.hours} год")
}
```

- мова добре масштабується від навчальних задач до enterprise
