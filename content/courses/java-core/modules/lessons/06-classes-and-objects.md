# Classes and Objects

До цього часу ми писали весь код у одному місці. Але справжні програми (наприклад, Minecraft чи Instagram) занадто великі для цього. Java використовує підхід, який називається **Об'єктно-Орієнтоване Програмування (ООП)**. Це спосіб написання коду, де ми уявляємо програму як набір віртуальних предметів (об'єктів), які взаємодіють між собою.

## Теорія

### Що таке Клас?

**Клас** — це креслення, шаблон або інструкція. 
Уяви завод, який випускає автомобілі. Креслення автомобіля (де вказано, що в нього є 4 колеса, двигун і колір) — це і є **Клас**. Само по собі креслення не може їздити, це просто схема на папері. 

Клас складається з двох частин:
1. **Поля (Attributes/Fields):** Характеристики. Який це об'єкт? (наприклад: ім'я, колір, швидкість).
2. **Методи (Methods):** Дії. Що цей об'єкт вміє робити? (наприклад: бігати, стрибати, видавати звук).

### Що таке Об'єкт?

**Об'єкт** — це реальна річ, створена за кресленням (класом). 
Якщо клас — це креслення автомобіля, то об'єкт — це реальна червона машина, яка стоїть у тебе у дворі. За одним кресленням (Класом) можна створити хоч тисячу машин (Об'єктів), і кожна з них може мати свій колір!

Щоб створити об'єкт у Java, ми використовуємо магічне слово **`new`**.

## Простий приклад

Давай створимо клас `Robot`, а потім в іншому місці створимо кількох реальних роботів!

```java
// Це наше креслення (Клас)
class Robot {
    // Поля (характеристики робота)
    String name;
    String color;
    int batteryLevel;

    // Методи (що робот вміє робити)
    public void sayHello() {
        System.out.println("Біп-боп! Привіт, я " + name + ". Мій колір " + color + ".");
    }

    public void work() {
        System.out.println(name + " працює...");
        batteryLevel = batteryLevel - 10; // Витрачаємо батарею
        System.out.println("Залишок батареї: " + batteryLevel + "%");
    }
}

// А це головний клас, де ми будемо запускати гру
public class Main {
    public static void main(String[] args) {
        // Створюємо ПЕРШОГО об'єкта-робота за допомогою слова 'new'
        Robot wallE = new Robot();
        wallE.name = "ВАЛЛ-І";
        wallE.color = "Жовтий";
        wallE.batteryLevel = 100;

        // Створюємо ДРУГОГО об'єкта-робота
        Robot eve = new Robot();
        eve.name = "ЄВА";
        eve.color = "Білий";
        eve.batteryLevel = 100;

        // Змушуємо їх діяти!
        wallE.sayHello();
        wallE.work();
        
        System.out.println("---");
        
        eve.sayHello();
    }
}
