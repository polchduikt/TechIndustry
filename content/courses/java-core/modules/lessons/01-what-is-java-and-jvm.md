# What is Java and JVM

Java — це обʼєктно-орієнтована мова програмування загального призначення.  
Вона широко використовується для створення веб-застосунків, серверних систем, мобільних додатків та enterprise-рішень.

---

## Мета уроку

Після цього уроку ти зможеш:

- пояснити, що таке Java
- розуміти принцип роботи JVM
- знати, як виконується Java-код
- орієнтуватися в екосистемі Java

---

## Теорія

### Що таке Java

**Java** — це строго типізована, компільована мова програмування, створена компанією Sun Microsystems у 1995 році.

Основні характеристики Java:

- обʼєктно-орієнтований підхід
- кросплатформеність
- автоматичне керування памʼяттю
- висока стабільність

Головний принцип Java:

> Write Once, Run Anywhere

Це означає, що програму можна написати один раз і запускати на будь-якій операційній системі, де встановлена JVM.

---

### Як працює Java

Процес виконання Java-програми відбувається у кілька етапів:

1. Код компілюється у байткод
2. Байткод виконується у JVM
3. JVM перетворює його у машинний код

Схема роботи:
Java Code → Compiler → Bytecode → JVM → Machine Code
---

### Що таке JVM

**JVM (Java Virtual Machine)** — це віртуальна машина, яка запускає Java-байткод.

JVM відповідає за:

- виконання коду
- керування памʼяттю
- безпеку
- оптимізацію виконання

Завдяки JVM Java є кросплатформеною.

---

### JDK, JRE та JVM

У Java є кілька важливих компонентів:

- **JVM** — виконує код
- **JRE (Java Runtime Environment)** — містить JVM та стандартні бібліотеки
- **JDK (Java Development Kit)** — набір інструментів для розробки

Простими словами:
JDK → включає JRE → включає JVM


---

### Де використовується Java

Java застосовується у різних сферах:

- Backend-розробка
- Android-додатки
- Банківські системи
- Великі корпоративні платформи
- Хмарні сервіси

Java часто використовують у великих системах, де важлива надійність.
