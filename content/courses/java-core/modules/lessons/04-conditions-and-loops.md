# 04-conditions-and-loops

## Мета уроку

Після цього уроку ти зможеш:

- керувати потоком виконання програми за допомогою `if-else`
- використовувати оператор множинного вибору `switch`
- створювати цикли `for`, `while` та `do-while`
- керувати циклами за допомогою `break` та `continue`

---

## Теорія

### Умовні оператори (if, else if, else)

Оператор `if` перевіряє умову (яка має дорівнювати `true` або `false`). Якщо умова виконується, код усередині блоку `{}` запускається.

- `if` — якщо умова істинна
- `else if` — додаткова умова, якщо попередня була хибною
- `else` — виконується, якщо жодна з умов не підійшла

---

### Оператор switch

Якщо тобі потрібно перевірити змінну на збіг з багатьма конкретними значеннями (наприклад, дні тижня від 1 до 7), зручніше використовувати `switch`, а не багато `if-else`.

---

### Цикли (Loops)

Цикли дозволяють виконувати блок коду задану кількість разів або поки виконується певна умова.

- **Цикл `while`**: перевіряє умову перед виконанням. Якщо умова `true`, виконує код. (Може не виконатися жодного разу).
- **Цикл `do-while`**: спочатку виконує код один раз, а вже потім перевіряє умову. (Завжди виконається хоча б раз).
- **Цикл `for`**: ідеально підходить, коли ти точно знаєш, скільки разів треба повторити дію. Має лічильник всередині себе.

---

### Переривання циклів (break та continue)

- `break` — негайно зупиняє і повністю виходить з циклу.
- `continue` — перериває поточну ітерацію (крок) циклу і одразу переходить до наступної.

---

## Простий приклад

```java
public class FlowControlDemo {
    public static void main(String[] args) {
        int score = 85;
        
        // Приклад умови
        if (score >= 90) {
            System.out.println("Оцінка: Відмінно");
        } else if (score >= 75) {
            System.out.println("Оцінка: Добре");
        } else {
            System.out.println("Оцінка: Задовільно");
        }
        
        System.out.println("---");
        
        // Приклад циклу for
        System.out.println("Рахуємо до 3:");
        for (int i = 1; i <= 3; i++) {
            System.out.println("Число: " + i);
        }
        
        System.out.println("---");
        
        // Приклад циклу while з break
        int counter = 1;
        while (true) { // Нескінченний цикл
            System.out.println("Спроба " + counter);
            if (counter == 2) {
                System.out.println("Зупиняємо цикл!");
                break; // Вихід з циклу
            }
            counter++;
        }
    }
}
