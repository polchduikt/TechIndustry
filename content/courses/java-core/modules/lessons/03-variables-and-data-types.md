# Variables and Data Types

Програми постійно працюють з даними: іменами користувачів, віком, цінами на товари або результатами обчислень. У Java для збереження цих даних використовуються змінні.

---

## Мета уроку

Після цього уроку ти зможеш:

- розуміти, що таке змінна та як її оголосити
- розрізняти примітивні типи даних у Java
- знати різницю між примітивними та посилальними типами
- правильно називати змінні (naming conventions)

---

## Теорія

### Що таке змінна

**Змінна** — це іменована комірка в пам'яті комп'ютера, де зберігається певне значення.
Java — це *строго типізована* мова. Це означає, що під час створення змінної ти обов'язково повинен вказати, який саме тип даних вона буде зберігати (наприклад, ціле число, текст чи дробове число).

Синтаксис оголошення:
`[Тип даних] [назва змінної] = [значення];`

---

### Примітивні типи даних

У Java є 8 примітивних типів даних. Вони зберігають прості значення і працюють дуже швидко:

**Цілі числа:**
- `byte` (від -128 до 127)
- `short` (від -32,768 до 32,767)
- `int` (стандартний вибір для цілих чисел, приблизно до 2 мільярдів)
- `long` (для дуже великих чисел, в кінці значення ставиться літера `L`)

**Дробові числа:**
- `float` (менша точність, в кінці ставиться `f`)
- `double` (стандартний вибір для дробових чисел, висока точність)

**Символи та логічні значення:**
- `char` (один символ, береться в одинарні лапки: `'A'`)
- `boolean` (логічний тип, може бути лише `true` або `false`)

---

### Посилальні типи (Reference Types)

Окрім примітивів, існують посилальні типи. Вони посилаються на складні об'єкти в пам'яті. Найвідоміший посилальний тип — це рядок тексту:

- `String` (текст, береться у подвійні лапки: `"Привіт"`)

> Зверни увагу: примітивні типи пишуться з малої літери (`int`, `double`), а класи та посилальні типи — з великої (`String`).

---

### Правила іменування (CamelCase)

У Java прийнято називати змінні стилем `camelCase` (верблюжий регістр) — перше слово з малої літери, а кожне наступне з великої, без пробілів.
Наприклад: `userAge`, `maxSpeed`, `totalAmountToPay`.

---

## Простий приклад

```java
public class VariablesDemo {
    public static void main(String[] args) {
        // Примітивні типи
        int age = 25;
        double height = 1.82;
        boolean isStudent = true;
        char grade = 'A';
        
        // Посилальний тип
        String name = "Олександр";
        
        // Виведення даних
        System.out.println("Ім'я: " + name);
        System.out.println("Вік: " + age);
        System.out.println("Зріст: " + height + " м");
        System.out.println("Студент: " + isStudent);
    }
}
