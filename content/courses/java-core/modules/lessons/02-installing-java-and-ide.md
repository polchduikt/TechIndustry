# Installing Java and IDE

Щоб почати програмувати на Java, недостатньо просто написати код у текстовому редакторі. Нам потрібно налаштувати робоче середовище: встановити інструменти для компіляції та зручну програму для написання коду.

---

## Мета уроку

Після цього уроку ти зможеш:

- завантажити та встановити Java Development Kit (JDK)
- налаштувати змінні середовища (за потреби)
- встановити та налаштувати середовище розробки (IDE)
- створити та запустити свій перший проект

---

## Теорія

### Що таке JDK і навіщо він потрібен

Як ми дізналися з попереднього уроку, для розробки потрібен **JDK (Java Development Kit)**. Він містить компілятор (`javac`), який перетворює твій код на зрозумілий для машини байткод, та JVM для його виконання.

Найпопулярніші версії JDK сьогодні — це Java 17 та Java 21 (LTS версії — довгострокова підтримка). Завантажити JDK можна з офіційного сайту Oracle або використовувати відкриті версії, такі як OpenJDK чи Amazon Corretto.

---

### Що таке IDE

**IDE (Integrated Development Environment)** — це інтегроване середовище розробки. Це спеціальна програма, яка полегшує написання коду, підсвічує синтаксис, автоматично виправляє дрібні помилки та дозволяє запускати програму однією кнопкою.

Найпопулярніші IDE для Java:

- **IntelliJ IDEA** (від JetBrains) — найпопулярніший вибір, стандарт індустрії. Для навчання ідеально підходить безкоштовна версія Community Edition.
- **Eclipse** — потужна безкоштовна IDE, часто використовується в старих корпоративних проектах.
- **Visual Studio Code** — легкий редактор, який за допомогою плагінів можна перетворити на повноцінну IDE для Java.

> Ми рекомендуємо використовувати IntelliJ IDEA Community Edition, оскільки вона найзручніша для новачків.

---

### Змінна середовища JAVA_HOME

Після встановлення JDK операційна система має знати, де саме знаходяться інструменти Java. Для цього в Windows/Linux/macOS налаштовується змінна середовища `JAVA_HOME`, яка вказує на папку зі встановленим JDK.
Сучасні IDE, такі як IntelliJ IDEA, часто роблять це автоматично.

---

## Простий приклад

Ось як виглядатиме твій перший проект у IntelliJ IDEA. Ти створюєш клас, додаєш головний метод `main` (з якого починається виконання будь-якої Java програми) і виводиш текст на екран.

```java
public class HelloWorld {
    // Метод main - це точка входу в програму
    public static void main(String[] args) {
        // Виводимо повідомлення в консоль
        System.out.println("Моє середовище розробки налаштовано!");
    }
}
